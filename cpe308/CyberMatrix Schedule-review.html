1.5.1. CyberMatrix Class Scheduler

CyberMatrix Class Schedular is a program used for student based scheduling as well as block scheduling and requirements based scheduling. In student based scheduling the scheduler is used on a student by student basis where students have there own unique class schedule. Lower level schools like elementary or middle school use block schedules inputting entire groups of students at a time.

CyberMatrix Class Scheduler provides all the tools needed to schedule students in various types of schools, but the program is not smooth or intuitive enough to be worthwhile.

Good Features:

  • Integrated several tools for scheduling in various formats such as individually or in a group

  • Several tools for auto scheduling individual students or groups to classes

  • Allows for data entry with files such as SQL tables or CSV files

  • Detailed option editing for scheduling, accounting for things like holidays, instructors, student discipline and student health.

Bad Features:

  • Search feature works very poorly

  • Toolbar of main menu is poorly laid out

  • Graphics at top of screen take away obscure toolbar buttons

  • Options for editing data are not organized very well, easy to get lost in the maze of possibilities

  • Lots of open ended features with little help form the help/faq menu

Missing Features:

  • more options for data entry other than just SQL tables or CSV files

  • multiple view options for the schedule calendar on the main view screen

  • editable toolbar, so the user can swap the tools in and out at their convenience
